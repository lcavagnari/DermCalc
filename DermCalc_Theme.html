/ 01 — UNDERTALE

Use the quiet register

Pure #FFFF00 on #000000 is the SAVE prompt — a peak moment, not a UI baseline. We use Snowdin's warm-dark and a tamed determination gold for everything that isn't the heart itself.

/ 02 — MATERIAL 3

Honest semantic names

No more Gold80 = #E2E2E2. Tokens map to roles: primary, surface-1/2, outline, severity.{mild,mod,severe}. Primary never collides with severity.

/ 03 — CLINICAL

Legibility above all

PASI scoring sessions are 5–10 minutes of focused reading. Body text on AA contrast (≥4.5:1), saturation under 70% except for severity, no pure-neon on pure-black.

/ 03 — SOUL ACCENTS

One soul per calculator — virtue as identity

The seven SOULs map to roles. Determination stays the brand. Each calculator gets the virtue that matches what the act of measuring asks of you. Severity reuses three of them on a clinical scale.
DETERMINATION
Brand · the heart
The user is the soul. Launcher icon, brand mark, primary CTA when no calculator owns the screen.
#E04848 · red
PATIENCE
BMI calculator
A slow measure. Weight and height drift across months. Patience is the virtue of long observation.
#4FB8B8 · cyan
BRAVERY
BSA calculator
Looking at the body honestly. Marking which regions are affected takes courage.
#E5853A · orange
INTEGRITY
PASI calculator
Sixteen scores across four districts. Integrity is rigor — scoring honestly even when it's tedious.
#4A6BD9 · blue
PERSEVERANCE
EASI calculator
Eczema is chronic. Tracking it over years is the refusal to give up.
#9B4FC4 · purple
KINDNESS
Profile · self-care
The patient is the subject. Kindness is remembering you're caring for a person, not a chart.
#5FB85F · green
JUSTICE
History · the record
The unflinching ledger. Every reading, when, with what severity. Justice is the truth of the timeline.
#E8B842 · yellow
↳ LAYERING RULE
Identity lives on screen chrome — the header strip, the active nav glyph, the screen-primary button.

Severity lives only on result tiles, using three souls as a universal scale:
Kindness = mild · Justice = moderate · Determination = severe.

A purple PASI screen with a green mild-result tile reads cleanly: the tile is the answer, the chrome is the room.

/ 07 — DROP-IN KOTLIN

Color.kt — rewrite

Tokens map 1:1 to the swatches above. Replace your current Color.kt with this; update Theme.kt to use the matching role.
package it.lcavagnari.pdm.dermcalc.ui.theme

import androidx.compose.ui.graphics.Color

// ============================================================

// Roles are honest. Names match what the value IS.
// ============================================================
